API Backend Login

Rutas disponibles en el backend:

http://127.0.0.1:5000/ Ruta base

  • POST /register Crear nuevo usuario
  • POST /login Iniciar sesión (devuelve token JWT)
  • GET /profile Ver perfil (requiere token)
  • GET /users Listar usuarios (requiere token)
  • PUT /users/<id> Editar usuario (requiere token)
  • DELETE /users/<id> Eliminar usuario (requiere token)

Base de datos: usermodule.db
